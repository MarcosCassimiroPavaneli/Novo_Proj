Central de Ajuda

Como este projeto funciona?

Este site é uma interface frontend clone do Gemini Advanced que foi criada para o seu projeto pessoal de inteligência artificial. Atualmente ela está configurada com respostas simuladas rápidas, mas já possui total suporte para se conectar diretamente com recursos do Azure OpenAI.

Dicas de Atalhos

  • Enter: Envia a mensagem digitada.
  • Shift + Enter: Insere uma nova linha na caixa de mensagem.
  • Botão de Nova Conversa: Limpa a tela e reseta o histórico de mensagens.

Sobre o Deploy

O repositório está linkado no GitHub e publicado diretamente no Vercel. Qualquer alteração empurrada com git push atualizará o site em produção instantaneamente.
